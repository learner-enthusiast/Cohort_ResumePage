# Arnab Samanta - Portfolio Website

A simple HTML portfolio website showcasing my professional experience, skills, projects, and contact information.

## Overview

This is a personal portfolio website built using pure HTML without any CSS styling. The website presents my professional profile as a Full Stack Developer, including my work experience, technical skills, open-source contributions, projects, education, and contact information.

## Features

- About Me section with professional summary
- Technical skills organized in lists
- Work experience with detailed achievements
- Open-source contributions
- Featured projects with live links
- Education background
- Contact information with social links
- Semantic HTML structure
- No external dependencies

## Technologies Used

- HTML5 (Pure semantic markup)
- No CSS or JavaScript

## Setup Instructions

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)

### Steps to Run

1. Clone the repository
   ```bash
   git clone https://github.com/learner-enthusiast/portfolio-html.git
   cd portfolio-html
   ```

2. Open the HTML file
   - Double-click on `index.html` to open in your browser
   - Or right-click and select "Open with" your preferred browser

### Alternative
- Download `index.html` and open it directly in any browser

## Usage

Open the HTML file in a browser and scroll through the sections:
- Header with name and title
- About Me section
- Technical Skills
- Professional Experience
- Open Source Contributions
- Projects (with live demo links)
- Education
- Contact Information

All external links open in new tabs for better user experience.

## Evaluation Parameters

### UI Layout and Structure
- Clear hierarchical structure using proper heading levels (h1, h2, h3, h4)
- Logical information flow from top to bottom
- Sections separated with horizontal rules
- Tables for structured data (Experience, Education)
- Lists for skills and achievements

### Spacing, Alignment, and Consistency
- Consistent heading usage throughout
- Uniform table structure with borders
- Horizontal rules between major sections
- Proper paragraph spacing

### Visual Clarity and Readability
- Descriptive section headings
- Bullet points for easy information scanning
- Tables with clear headers
- Strong tags for important labels
- Descriptive link text

### Proper Use of HTML Semantics
- Correct heading hierarchy (h1 for name, h2 for sections, h3-h4 for subsections)
- Unordered lists for skills and achievements
- Tables with thead and tbody structure
- Anchor tags with target="_blank" for external links
- Horizontal rules for section breaks
- Paragraph tags for text content
- Strong tags for emphasis
- Proper document structure with DOCTYPE, html, head, and body tags

### Overall User Experience and Design Quality
- Easy navigation with clear headings
- No JavaScript required
- External links open in new tabs
- Logical information organization
- Accessible contact information
- Professional content presentation
- Works on all devices
